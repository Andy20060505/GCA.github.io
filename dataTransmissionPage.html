Simplex Data Transmission

Simplex data transmission is in one-direction only(from sender to receiver). It allows for data to be sent in only one direction.

Examples: Sending data from the keyboard to the CPU.

Half Duplex Data Transmission

Half-duplex data transmission is in both directions but not at the same time(data can be sent from 'A' to 'B' or from 'B' to 'A' along the same line ,but not at the same time).It allows data to travel in both directions, however, it cannot occur simultaneously.This means that data can only travel one way at a time.

Examples: Walkie talkies only allow communication one way at a time.
